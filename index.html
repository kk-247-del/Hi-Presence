E2EE Engineering

Security & The Web Bridge

Whether on a Native App or a Web Session, the Locus Class constraints remain absolute.

  • Key-by-Key E2EE: Every character of Live Text is encrypted before it leaves the browser or the app.
  • Always-Warm Voice: P2P voice streaming is encrypted end-to-end, bypassing House of Ariaka servers entirely.
  • Session Purge: Web sessions utilize volatile browser memory; once the tab closes or the moment collapses, the E2EE keys are wiped from existence.
