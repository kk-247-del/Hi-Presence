Locus Class | Moment-Based Engineering

Reject the Archive.

Presence is a fundamentally different class of communication. It rejects the metaphor of messaging as exchange and instead treats interaction as co-presence. It is an environment where moments occur only during shared focus and dissolve the instant conditions fail. No storage. No history. Only the now.

Active Law 01

Presence Detection

The system continuously evaluates foreground activity and device state. If presence becomes ambiguous, the connection ceases.

Active Law 02

Live Text Streaming

Characters are streamed in real-time as they are formed. Interaction is a live event, not a package delivered after the fact.

Active Law 03

Moment Collapse

Our MomentEngine™ enforces a total teardown if focus is lost. There is no pause, no recovery, and no resumption.

Architecture & Compliance

Privacy by Constraint

In the Locus Class, privacy is not a setting; it is a structural impossibility of data persistence.

  • Zero Local Disk Writes: Interaction data lives only in volatile memory (RAM) and is purged instantly upon moment collapse.
  • Voice Escalation Security: Real-time audio streams are peer-to-peer (WebRTC) and are never recorded or processed by Cariselle servers.
  • Identity Verification: Access is evaluated continuously based on live device state. Presence is the only credential.
  • Hardware Compliance: The app utilizes standard Windows APIs for Microphone and Camera access only during active, foregrounded moments.
